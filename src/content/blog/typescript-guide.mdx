# TypeScript完全ガイド

TypeScriptは、JavaScriptに静的型付けを追加した言語です。大規模なアプリケーション開発において、その威力を発揮します。

## TypeScriptの基本

### 基本的な型

```typescript
// 基本型
let name: string = "Portfolio";
let age: number = 25;
let isActive: boolean = true;

// 配列
let numbers: number[] = [1, 2, 3];
let names: Array<string> = ["Alice", "Bob"];
```

### インターフェース

```typescript
interface User {
  id: number;
  name: string;
  email: string;
  isActive?: boolean; // オプショナル
}

const user: User = {
  id: 1,
  name: "John",
  email: "john@example.com"
};
```

### ジェネリクス

```typescript
function identity<T>(arg: T): T {
  return arg;
}

let output = identity<string>("Hello");
```

## 実践的な活用法

### React との組み合わせ

```typescript
interface Props {
  title: string;
  children: React.ReactNode;
}

const Component: React.FC<Props> = ({ title, children }) => {
  return (
    <div>
      <h1>{title}</h1>
      {children}
    </div>
  );
};
```

## まとめ

TypeScriptを使うことで、より安全で保守性の高いコードが書けるようになります。最初は学習コストがありますが、長期的には開発効率が大幅に向上します。
